ADFC Falcon software installation

 

 

Click on the link to download the file and save the “ADFC Falcon.exe” file in your computer. After the download is completed locate the “ADFC Falcon.exe” and click on it, you will get a security warning window, click on “Run” (see below pic)

 

 

 

When you click on run, the installation options window will open, Click on “Next”.

(See below pic)

 

 

After you click on next, an installation folder will open, click on “Install” to start the installation (see below pic)

 

 

After the installation is completed, check your desk top, you will see the below

Icon

Click on the icon to start ADFC Falcon. Only when you run the software for the first time, you will be asked to insert to username and password twice.
